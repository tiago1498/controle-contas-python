# Sistema de Controle de Contas em Python

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação,
uso de funções, dicionários, estruturas de repetição e tratamento de erros.

## Funcionalidades
- Cadastrar contas
- Atualizar valor de contas
- Pagar contas (remover)
- Listar contas cadastradas
- Somar o valor total das contas
- Validação de entradas do usuário

## Tecnologias Utilizadas
- Python 3
- Estruturas de controle (if/elif/else)
- Laço de repetição (while)
- Funções
- Dicionários
- Tratamento de exceções (try/except)

## Como executar
1. Tenha o Python 3 instalado
2. Execute o arquivo:
```bash
python sistema_contas.py
